# geophp
[GeoPHP](https://geophp.net/) Symfony2 Bundle

## Install

Repository currently marked as dev. Set up your `composer.json` consequently:

```json
{
    "require": {
        "pbald/geophp-bundle": "*@dev"
    },
    "repositories": [
      {
        "type": "vcs",
        "url": "https://github.com/pbald/geophp-bundle.git"
      }
    ]
}
```
